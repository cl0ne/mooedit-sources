tempdir()

moo.tempdir()

Return the path of medit temporary file directory. This directory is created automatically and is removed on exit. It is unique among all medit instances.

Returns: string

tempnam()

moo.tempnam(extension=nil)

Generate a unique filename for a temporary file. Generated filename is located inside directory returned by tempdir(), and it will be automatically removed on exit.

extension :

string or nil

Returns: string
